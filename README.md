## A demo realization of LSTM time series prediction in PyTorch with simple data

This project implements LSTM on time series prediction with a complete pipeline of dataset split, feature scaling, as well as LSTM network training and evaluation using PyTorch. 
The dataset is read from an online .csv file, with the single feature of the number of airline passengers across 144 days.

It is a practice demo reproduced and modified based on the link: https://machinelearningmastery.com/lstm-for-time-series-prediction-in-pytorch/.
